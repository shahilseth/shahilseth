<h2 align="left">🚀 Featured Projects</h2>

**[CodeSheriff](https://github.com/shahilseth/codesheriff)** — Multi-Agent Codebase Intelligence
- 4-agent pipeline: Navigator (AST-based chunking via tree-sitter) → Analyst → Critic → Synthesiser
- Answers natural-language questions about any GitHub repo — retrieves relevant code, reasons about design intent, adversarially critiques that reasoning
- Full eval suite: **0.80 retrieval recall**, **0.90 confidence calibration**
- Stack: FastAPI · Groq/Llama 3 · ChromaDB · HuggingFace Embeddings · React/Vite

**[PaperTrail](https://github.com/shahilseth/papertrail)** — GraphRAG over Academic Literature
- Answers multi-hop citation questions that vector search structurally can't handle
- Neo4j knowledge graph + vector retrieval hybrid with a query planner that routes by question type
- Stack: Python · Neo4j · ChromaDB · FastAPI · Groq

**[Nexus](https://github.com/shahilseth/nexus)** — Team Task Orchestration Platform
- Full-stack platform built for AI-first teams: RBAC, kanban, real-time WebSocket updates
- Live at [nexus-frontend-production-e212.up.railway.app](https://nexus-frontend-production-e212.up.railway.app)
- Stack: Next.js 14 · Express · TypeScript · PostgreSQL/Supabase · Railway

**[DocSense](https://github.com/shahilseth/docsense)** — RAG-Powered Document Q&A
- PDF question answering with citations and eval scores
- Stack: LangChain · pgvector · Supabase · FastAPI
